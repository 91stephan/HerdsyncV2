---------------------------------------------------------------------
TRAINING THAT WAS CONDUCTED
---------------------------------------------------------------------

TRAINING PROGRAMME 1: System Administration
Duration: 3 days
Audience: Ministry IT Department (4 staff), hosted at Maseru
Topics covered:
  Day 1 — Infrastructure management
    - Docker Compose service management (start, stop, update)
    - Database backup and restore procedure (backup.sh)
    - Server monitoring (disk space, container health, logs)
    - How to apply new database migrations
    - SSL certificate renewal (nginx.conf)
  Day 2 — Supabase Studio administration
    - Using Supabase Studio to inspect database
    - Managing user accounts and roles (profiles table)
    - Reviewing the audit_log table
    - Managing offline_sync_queue conflicts
    - LIAMIS and MIS sync log monitoring
  Day 3 — Security and incident response
    - UFW firewall rules management
    - Fail2ban log review
    - Responding to a data breach (audit log + account suspension)
    - Supabase Edge Function deployment
    - How to roll back a bad migration

TRAINING PROGRAMME 2: Center Manager Operations
Duration: 2 days
Audience: Breeding center managers (5 centers, 1 manager each)
  - Semonkong Merino Stud Center
  - Mohale's Hoek Angora Breeding Center
  - Quthing National Stud Center
  - Mokhotlong Highland Breeding Center
  - Leribe Fiber Production Center
Topics covered:
  Day 1 — Daily operations
    - Registering animals and fiber metrics
    - Managing breeding records and birthing outcomes
    - Running the genetic indices recalculation
    - Using the Culling & Exchange workflow
    - Reviewing reproductive indices for their district
  Day 2 — Reporting and compliance
    - Creating WOAH disease reports
    - Exporting WAHIS-compliant JSON for Ministry submission
    - Generating EFI fiber export for Basotho Brand reporting
    - Document vault management
    - Training their field workers on offline sync

TRAINING PROGRAMME 3: Field Worker Orientation
Duration: 1 day (delivered at each center)
Audience: Extension workers (estimated 15 across all centers)
Topics covered:
  - Using the mobile app offline
  - Registering animals with RFID scanner
  - Recording culling exchange events
  - Understanding the Sync Status Bar
  - Escalating conflicts to district officer

---------------------------------------------------------------------
ASSESSMENT CONDUCTED
---------------------------------------------------------------------
Each participant completed a practical assessment:
  IT Staff: Successfully perform a database backup and restore
  Center Managers: Register an animal, create a culling record,
    generate a genetic index report
  Field Workers: Register 3 animals offline and sync on reconnect

Pass rate: 18/19 participants (95%)
One field worker (Mokhotlong) requires a repeat session due to
connectivity challenges during training.
